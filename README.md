# PortalSemed-DF
Repositório dedicado à disciplina de Projeto Extensionista Integrador I, voltado para o MVP 

## Como iniciar o servidor local

Use o roteador raiz para servir corretamente a pasta `PortalSemed/public`:

```powershell
php -S localhost:8000 router.php
```

A partir daí, acesse a aplicação em `http://localhost:8000`.
